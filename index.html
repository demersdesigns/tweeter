About Tweeter

What is Tweeter?

Tweeter is a very simple Twitter viewer demo application. This application was produced as a proof of concept and thus has a very limited scope.

What Can Tweeter Do?

The Tweeter application allows you to view the latest 15 tweets authored by the last 20 people you followed on Twitter. It is intentionally limited in its feature set and is hardcoded to my personal Twitter account.

How Was Tweeter Built?

The HTML5 and CSS3 are based upon the HTML5 Boilerplate project.

The application uses semantic.gs for it's grid system and is a percentage-based layout with a max-width of 940px. This lends itself nicely to responsive design if there were a few media queries added.

CSS was written using the LESS CSS framework and compiled using the LESS.app application.

The site is modeled with MVVM in mind and utilizes the Knockout.js framework for local model storage as well as templating.
